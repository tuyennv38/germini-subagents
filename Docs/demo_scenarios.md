# Demo: Xây dựng tính năng đơn giản

Bạn có thể chạy thử các kịch bản sau để thấy cách "Team AI" làm việc:

### Kịch bản 1: Tạo một ứng dụng Todo List cơ bản
**Yêu cầu**: "Tôi muốn xây dựng một ứng dụng Todo List đơn giản bằng NodeJS."

**Cách Subagent thực hiện**:
1. **PM**: Tạo `task.md` với danh sách: setup project, viết UI, viết logic lưu trữ.
2. **Architect**: Đề xuất sử dụng cấu trúc file gọn gàng.
3. **Developer**: Chạy `npm init`, tạo `index.js`, viết code.
4. **QA**: Chạy thử script bằng node để kiểm tra lỗi syntax.

### Kịch bản 2: Debug và tối ưu hóa
**Yêu cầu**: "Kiểm tra file code hiện tại và tối ưu hóa hiệu suất."

**Cách Subagent thực hiện**:
1. **Developer**: Đọc file, tìm đoạn code chưa tối ưu.
2. **QA**: Thực hiện đo thời gian chạy trước và sau khi sửa.
